Each project lives in its own folder with its own `index.html`, `style.css`, and `script.js`.

## 🚀 Running Locally

1. Clone the repo
```bash
   git clone https://github.com/Andrew-Fernando-15/Projects.git
```
2. Open any project's `index.html` in your browser — no build step required
